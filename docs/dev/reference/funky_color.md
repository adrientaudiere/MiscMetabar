# Funky palette color

Funky palette color

## Usage

``` r
funky_color(n)
```

## Arguments

- n:

  a number of colors

## Value

a color palette

## See also

The R package RColorBrewer, proposing a nice selection of color
palettes. The viridis package, with many excellent palettes

## Author

Thibaut Jombart in `adegenet` package

## Examples

``` r
funky_color(5)
#> [1] "#A6CEE3" "#52AF43" "#F06C45" "#B294C7" "#B15928"
```
