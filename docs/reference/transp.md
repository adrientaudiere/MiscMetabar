# Adds transparency to a vector of colors

Adds transparency to a vector of colors

## Usage

``` r
transp(col, alpha = 0.5)
```

## Arguments

- col:

  a vector of colors

- alpha:

  (default 0.5) a numeric value between 0 and 1 representing the alpha
  coefficient; 0: total transparency; 1: no transparency.

## Value

a color vector

## See also

The R package RColorBrewer, proposing a nice selection of color
palettes. The viridis package, with many excellent palettes
